# Отправка на сервер

## На локальной машине

```bash
# 1. Закоммитить и отправить в репозиторий
git add .
git status   # проверь список файлов
git commit -m "feat: часовой пояс пользователя, вся логика по времени в поясе пользователя"
git push origin main
```

## На сервере (по SSH)

```bash
# 2. Подключиться к серверу
ssh root@YOUR_SERVER_IP

# 3. Перейти в директорию бота
cd /root/forex-bot

# 4. Обновить код
git pull origin main

# 5. Установить зависимости (если менялся package.json)
npm install

# 6. Собрать
npm run build

# 7. Перезапустить бота
pm2 restart forex-bot

# 8. Проверить логи
pm2 logs forex-bot --lines 50
```

## Проверка после деплоя

- В Telegram: `/settings` → кнопка **«Часовой пояс»** → выбор города (например Нью-Йорк).
- Время в `/daily` и `/tomorrow` отображается в выбранном поясе.
- Тихий режим и утренняя рассылка в 08:00 — в поясе пользователя.
